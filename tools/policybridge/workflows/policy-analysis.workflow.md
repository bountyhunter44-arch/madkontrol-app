# Policy Analysis Workflow

1. Capture raw policy input and source metadata.
2. Identify goals, tools, target groups, and funding assumptions.
3. Separate facts, estimates, values, and assumptions.
4. Run consequence analysis conceptually.
5. Run financing realism checklist.
6. Run implementation risk checklist.
7. Produce public explanation draft.
8. Label uncertainty.
9. Require human review before public or high-impact use.
