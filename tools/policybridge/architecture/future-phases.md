# Future Phases

## Phase 0: Foundation

Current scope:

- Docs.
- Registries.
- Governance boundaries.
- Workflow definitions.
- AI profiles.
- Public explanation templates.

## Phase 1: Static Prototype

Possible future scope:

- Static policy analysis mockups.
- Example scenario cards.
- Example compromise maps.
- No backend.

Requires explicit approval.

## Phase 2: Text-Based Analysis Runtime

Possible future scope:

- Upload or paste policy text.
- Produce structured analysis.
- Generate public explanation draft.
- Human review workflow.

Requires data, privacy, neutrality, and source governance review.

## Phase 3: Scenario Engine

Possible future scope:

- Timeline feasibility checks.
- Financing realism checks.
- Stakeholder impact summaries.
- Multiple assumptions per scenario.

Requires model validation and uncertainty design.

## Phase 4: Public Trust Layer

Possible future scope:

- Public explanations.
- Correction history.
- Source disclosure.
- Human review status.

## Phase 5: Realtime Deliberation

Possible future scope:

- Live debate support.
- Realtime synthesis.
- Facilitated citizen panels.

Not part of current scope.
