# Public Debate Safety

## Goal

Support robust disagreement without abuse or manipulation.

## Safety Checks

- Does the analysis attack people rather than policies?
- Does it omit uncertainty?
- Does it imply a party preference without saying so?
- Does it cite facts without sources?
- Does it confuse values with empirical claims?
- Does it hide who is affected?
- Does it present forecasts as guarantees?

## Intervention Levels

1. Clarify.
2. Ask for evidence.
3. Add uncertainty label.
4. Add alternative perspective.
5. Escalate to human review.
6. Block public use until reviewed.
