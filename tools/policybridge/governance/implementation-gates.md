# Implementation Gates

No runtime is allowed in this phase.

## Gate 1: Source and Evidence Policy

Required before any AI analysis can cite, rank, or compare evidence.

## Gate 2: Neutrality Review

Required before public analysis is generated.

## Gate 3: Privacy and Data Retention

Required before storing inputs, outputs, user sessions, or public corrections.

## Gate 4: Model and Prompt Governance

Required before integrating any model provider.

## Gate 5: Simulation Validation

Required before presenting scenario outcomes as decision support.

## Gate 6: Public Communication Review

Required before citizen-facing explanations are published.

## Gate 7: Protected System Approval

Required before touching auth, payment, Firestore rules, onboarding, profiles, task engine, or deployment.
