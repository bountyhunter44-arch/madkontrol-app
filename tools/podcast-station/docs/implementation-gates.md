# Implementation Gates

No implementation is included in this phase. Future implementation must pass these gates.

## Gate 1: Privacy and Consent

Required before storing transcripts, voices, participant metadata, or public episode material.

## Gate 2: Source and Evidence Policy

Required before AI summaries can cite or rank evidence.

## Gate 3: Moderation Governance

Required before automated moderation can affect public output.

## Gate 4: AI Runtime Selection

Required before integrating model providers or realtime systems.

## Gate 5: Data Persistence

Required before adding database collections, schemas, indexes, or retention policies.

## Gate 6: Identity and Payment

Required before touching auth, Stripe, onboarding, subscriptions, or user profiles.

## Gate 7: Deployment

Required before any hosting, functions, or CI/CD change.
