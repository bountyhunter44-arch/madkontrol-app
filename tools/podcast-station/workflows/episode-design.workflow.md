# Episode Design Workflow

1. Define the public question.
2. Identify the debate mode: exploratory, deliberative, adversarial, or synthesis.
3. Identify human participant roles.
4. Select AI profiles from `registries/ai-profile-registry.json`.
5. Define what evidence is allowed and what remains unverified.
6. Define moderation risks.
7. Define summary outputs.
8. Define transparency labels.
9. Stop before implementation unless a future task explicitly approves runtime work.
