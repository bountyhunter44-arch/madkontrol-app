# Public Summary Review Workflow

1. Check that all major positions are represented.
2. Check that disagreement is preserved.
3. Check that uncertainty is explicit.
4. Check that AI-generated text is labeled.
5. Check that evidence and inference are separated.
6. Check that compromise candidates are marked as drafts.
7. Check that public correction pathways are described.
8. Require human review for high-impact topics before publication.
