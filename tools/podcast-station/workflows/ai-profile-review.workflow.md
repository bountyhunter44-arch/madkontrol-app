# AI Profile Review Workflow

1. Confirm the profile has a narrow role.
2. Confirm the profile discloses uncertainty.
3. Confirm the profile cannot claim lived experience.
4. Confirm the profile cannot invent sources.
5. Confirm the profile cannot declare final truth.
6. Confirm the profile improves human reasoning rather than replacing it.
7. Register the profile in `registries/ai-profile-registry.json`.
8. Add or update the matching file in `profiles/`.
