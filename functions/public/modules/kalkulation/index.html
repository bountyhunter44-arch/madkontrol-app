Hvad viser siden?

Denne side viser kostpris på opskrifter baseret på ingredienser, mængder og indkøbspriser. Den er bygget som fundament for senere prisstyring og avanceret kalkulation.

  • Kostpris pr. opskrift
  • Pris pr. ingredienslinje
  • Antal ingredienser i opskriften
  • Klar til salgspris og foodcost % senere

Næste skridt

Når denne side er godkendt, kan vi koble den til dine masterfiler og senere til Firestore, så kalkulationen bliver drevet af rigtige data og ikke kun frontend-eksempler.

  • Indlæsning fra recipes.json
  • Indlæsning fra ingredients.json
  • Prisberegning via ingredient-suppliers.json
  • Salgspris, avance og dækningsbidrag
